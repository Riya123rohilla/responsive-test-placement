
# Responsive Frontend Assessment

This project was created as part of a frontend development assessment. The task was to build a responsive landing page using HTML, CSS, and JavaScript while following modern UI practices and accessibility standards.

The project mainly focuses on responsive layouts, slider functionality, product cards, and interactive tabs.

---

## What I Built

### Hero Banner Slider
I created a responsive banner slider with:
- autoplay
- navigation arrows
- pagination dots
- infinite loop
- swipe support for mobile users

### Product Grid Section
The product section contains responsive product cards with:
- brand name
- product title
- current price
- original price

The layout automatically adjusts for desktop, tablet, and mobile screens.

### Tabs / Accordion
I also created an interactive tabs section for desktop view and accordion functionality for mobile view.

---

## Technologies Used

- HTML5
- CSS3
- JavaScript ES6
- Swiper.js

---

## Responsive Design

The UI is designed for:
- Desktop
- Tablet
- Mobile devices

Media queries were used to handle responsiveness.

Some accessibility practices included in the project:
